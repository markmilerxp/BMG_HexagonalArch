🛡️ BMG Seguros – Ajuda do Sistema

Sistema de gerenciamento de Propostas e Contratações de seguros.

1. Visão geral

O sistema permite criar propostas de seguro, alterar seu status e contratar as propostas aprovadas. A comunicação com o backend usa API REST e as regras de negócio estão descritas abaixo.

2. Status das propostas

Cada proposta possui um dos quatro status abaixo. As transições permitidas são controladas.

StatusValorDescrição
Em Análise1Status inicial ao criar a proposta
Aprovada2Proposta aprovada e pode ser contratada
Rejeitada3Proposta rejeitada
Contratada4Proposta já contratada (status final)

3. Regras de transição de status

  • Em Análise → pode ir para Aprovada ou Rejeitada
  • Aprovada → pode ir para Contratada (somente pela tela de Contratações, ao confirmar)
  • Rejeitada → não pode mudar para outro status
  • Contratada → não pode mudar para outro status

4. Tela Propostas

  • Criar proposta: preencha Nome do Cliente e Valor da Cobertura. A proposta é criada com status "Em Análise".
  • Alterar status: use o botão de edição na linha e escolha o novo status (apenas transições permitidas).
  • Excluir: só é possível excluir proposta com status diferente de Contratada.
  • Busca: filtre por ID da proposta, nome do cliente ou status (incluindo texto do status, ex.: "Em Análise", "Aprovada").

5. Tela Contratações

  • São listadas todas as propostas, com qualquer status.
  • Os três campos da contratação (PropostaId, Data Contratação, Nº Contrato) são preenchidos quando a proposta já foi contratada.
  • O botão Confirmar só é habilitado para propostas com status Aprovada. Ao confirmar, a proposta passa a "Contratada" e é gerado o número do contrato.
